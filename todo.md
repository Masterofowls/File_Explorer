# Fluent File Explorer - Future Tasks & Roadmap

## Core Functionality

- [ ] Valid full exit (graceful shutdown with cleanup)
- [ ] Instant code runner (execute scripts/binaries from context menu)
- [ ] Working native terminal (integrated terminal with proper cleanup)
- [ ] File properties full control (detailed properties dialog with edit capabilities)
- [ ] Unlock right-click items selection (allow selecting items via right-click without context menu interference)

## UI/UX Improvements

- [ ] Theme selection (light/dark/custom themes)
- [ ] Icon theme selection (multiple icon packs)
- [ ] Font selection (font family and size options)
- [ ] More adaptive layout (expanding elements by mouse drag, not just manual settings)
- [ ] Remove background terminal tasks 1-second pop-ups (clean up terminal notifications)

## File Operations & Archive Support

- [ ] More zip/unzip options (compression level, password protection, etc.)
- [ ] File size compression option (non-archive compression tools)
- [ ] More context menu options (custom actions, script execution)

## Search & Navigation

- [ ] More search options (regex, size filters, date range, etc.)
- [ ] Integration with Everything search (Windows search integration)

## Media & Preview

- [ ] More file formats preview support (video, audio, code, documents)
- [ ] Enhanced preview panel with codec information

## Settings & Preferences

- [ ] Remember user preferences (last used options, window state, layout)
- [ ] Multiple language support (i18n implementation)
- [ ] Settings persistence (local storage or config file)
- [ ] Customizable keyboard shortcuts

## Distribution & Deployment

- [ ] Full installation runtime for deep integration (Windows shell integration, file type associations)
- [ ] Lightweight web online version for quick demonstration
- [ ] Working updates (auto-update mechanism with version checking)
- [ ] Cross-platform distribution (DMG for macOS, Snap/AppImage for Linux, MSIX for Windows)

## Performance & Optimization

- [ ] Lazy loading for large directories
- [ ] Virtual scrolling for file lists
- [ ] Caching mechanism for frequently accessed paths
- [ ] Background indexing for faster search
- [ ] Memory optimization for large files

## Additional Features

- [ ] Batch file operations (bulk rename, bulk move)
- [ ] File tagging system
- [ ] Favorites/bookmarks enhancements
- [ ] Quick access customization
- [ ] Duplicate file finder
- [ ] File integrity checker (hash verification)
- [ ] Cloud storage integration (OneDrive, Google Drive, etc.)
